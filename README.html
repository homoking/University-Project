🎓 سامانه مدیریت دانشگاه

یک پروژه کامل تحت وب برای مدیریت دانشجویان، اساتید، دروس و گزارشات دانشگاهی.

🧩 ساختار پروژه


📁 project-root
│
├── index.html            # صفحه اصلی
├── login.html            # صفحه ورود به سیستم
├── students.html         # فرم ثبت و مشاهده اطلاعات دانشجویان
├── teachers.html         # فرم ثبت و مشاهده اطلاعات اساتید
├── courses.html          # فرم ثبت دروس
├── reports.html          # گزارشات مدیریتی
│
├── main.py               # API بک‌اند با FastAPI
├── repo.js               # اسکریپت اصلی فرانت‌اند

🚀 تکنولوژی‌های استفاده‌شده

بخشابزار/فریمورک
FrontendHTML, Tailwind CSS, JavaScript, Toastify, Chart.js
BackendPython, FastAPI, SQLAlchemy, Pydantic
DatabaseSQLite
فونتVazirmatn (گوگل فونت)

🧠 قابلیت‌ها

  • 🔐 صفحه ورود (Login)
  • 🧑‍🎓 ثبت و مدیریت دانشجویان
  • 🧑‍🏫 ثبت و مدیریت اساتید
  • 📚 ثبت دروس و اتصال آن‌ها به اساتید
  • 📊 گزارشات بصری و جدول‌محور
  • 🔍 فیلتر و جستجو بر اساس دانشکده، رشته و نام
  • ✅ اعتبارسنجی دقیق در سمت سرور

🛠 راه‌اندازی پروژه

  1. نصب وابستگی‌ها: 
    pip install fastapi uvicorn sqlalchemy pydantic
  2. اجرای سرور: 
    uvicorn main:app --reload
  3. باز کردن فایل HTML با Live Server یا مرورگر
