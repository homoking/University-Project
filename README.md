# 🎓 سامانه مدیریت دانشگاه

سامانه مدیریت دانشگاه یک پلتفرم تحت وب مدرن برای مدیریت جامع فرآیندهای دانشگاهی است. این پروژه با هدف خودکارسازی عملیات آکادمیک طراحی شده و امکان مدیریت دانشجویان، اساتید، دروس، و گزارشات را با رابط کاربری بصری و API قدرتمند فراهم می‌کند. با استفاده از فناوری‌های پیشرفته مانند FastAPI، Tailwind CSS، و Docker، این سامانه عملکرد بالا، امنیت، و مقیاس‌پذیری را تضمین می‌کند.

## 🚀 راهنمای نصب جامع

### پیش‌نیازهای سخت‌افزاری
| کامپوننت | حداقل نیاز | توصیه شده |
|----------|------------|-----------|
| CPU | 2 Core | 4 Core |
| RAM | 4GB | 8GB |
| فضای ذخیره‌سازی | 50GB | 100GB |

### نصب با Docker (پیشنهادی)
```bash
# کلون پروژه
git clone https://github.com/your-repo/university-system.git
cd university-system

# تنظیم محیط
cp .env.example .env
nano .env  # ویرایش تنظیمات

# اجرای سرویس‌ها
docker-compose up -d --build
```

### نصب دستی (توسعه)
```bash
# نصب بک‌اند
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# اجرای سرور
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

## 📖 راهنمای کاربری کامل

### ورود به سیستم
1. به آدرس `https://your-domain/login` مراجعه کنید
2. اطلاعات کاربری را وارد نمایید
3. از منوی اصلی ماژول مورد نظر را انتخاب کنید

### مدیریت دانشجویان
- **ثبت دانشجو جدید**:
  1. به بخش دانشجویان بروید
  2. روی دکمه "ثبت جدید" کلیک کنید
  3. فرم را کامل پر کنید
  4. اطلاعات را ذخیره نمایید

### گزارش‌گیری
- **گرفتن گزارش ترمی**:
  1. به بخش گزارشات بروید
  2. بازه زمانی را انتخاب کنید
  3. نوع گزارش را مشخص کنید
  4. گزارش را دریافت یا چاپ کنید

## 📚 مستندات فنی API

### Endpoint‌های اصلی
| متد | مسیر | توضیحات |
|------|-------|----------|
| GET | /api/students | لیست دانشجویان |
| POST | /api/students | ایجاد دانشجوی جدید |
| GET | /api/students/{id} | اطلاعات دانشجوی خاص |

### نمونه درخواست
```bash
curl -X GET "https://api.your-domain.com/api/students" \
  -H "Authorization: Bearer your-token"
```

### پاسخ نمونه
```json
{
  "data": [
    {
      "id": 1,
      "name": "علی محمدی",
      "student_id": "99123456",
      "major": "مهندسی کامپیوتر"
    }
  ],
  "count": 1
}
```
