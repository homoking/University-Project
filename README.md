🎓 سامانه مدیریت دانشگاه
سامانه مدیریت دانشگاه یک پلتفرم تحت وب پیشرفته برای مدیریت فرآیندهای دانشگاهی شامل اطلاعات دانشجویان، اساتید، دروس، و گزارشات است. این پروژه با استفاده از فناوری‌های مدرن توسعه داده شده و با تمرکز بر تجربه کاربری ساده، عملکرد بالا، و اعتبارسنجی دقیق طراحی شده است. هدف این سامانه، خودکارسازی و بهبود مدیریت داده‌های دانشگاهی با ارائه رابط کاربری بصری و API قدرتمند است.
این مستند راهنمای جامعی برای نصب، استفاده، توسعه، و نگهداری پروژه ارائه می‌دهد. چه یک توسعه‌دهنده باشید که قصد مشارکت در پروژه را دارد، چه یک مدیر سیستم که می‌خواهد سامانه را مستقر کند، یا یک کاربر نهایی که از امکانات آن استفاده می‌کند، این سند شما را راهنمایی خواهد کرد.

🏫 معرفی پروژه
سامانه مدیریت دانشگاه یک راه‌حل جامع برای مدیریت فرآیندهای آکادمیک است که با هدف ساده‌سازی عملیات دانشگاهی طراحی شده است. این سامانه امکان ثبت، ویرایش، حذف، و جستجوی اطلاعات دانشجویان، اساتید، و دروس را فراهم می‌کند و گزارشات بصری و آماری برای تصمیم‌گیری مدیریتی ارائه می‌دهد. با استفاده از فناوری‌های مدرن مانند FastAPI برای بک‌اند و Tailwind CSS و JavaScript برای فرانت‌اند، این پروژه عملکرد بالا، رابط کاربری جذاب، و امنیت را تضمین می‌کند.
این پروژه به‌گونه‌ای طراحی شده که برای دانشگاه‌های کوچک تا متوسط مناسب باشد و قابلیت مقیاس‌پذیری برای افزودن امکانات جدید را دارد. همچنین، با استفاده از Docker، استقرار و نگهداری آن ساده و قابل اعتماد است.

🛠 معماری و تکنولوژی‌ها
این پروژه از معماری کلاینت-سرور استفاده می‌کند، با جداسازی کامل بک‌اند و فرانت‌اند برای انعطاف‌پذیری و مقیاس‌پذیری. بک‌اند با FastAPI پیاده‌سازی شده و از SQLite به‌عنوان پایگاه داده استفاده می‌کند. فرانت‌اند با HTML، Tailwind CSS، و JavaScript توسعه یافته و از کتابخانه‌هایی مانند Axios و Chart.js بهره می‌برد.

نصب با Docker

کلون کردن پروژه:
git clone https://github.com/your-repo/university.git
cd university


ایجاد پایگاه داده اولیه:
touch university.db
sqlite3 university.db

INSERT INTO departments (name) VALUES ('فنی مهندسی');
INSERT INTO departments (name) VALUES ('علوم پایه');
INSERT INTO departments (name) VALUES ('اقتصاد');
.exit


اجرای Docker Compose:
docker-compose up --build -d


دسترسی به سامانه:

رابط کاربری: https://harmakan.ir یا http://localhost
API: http://localhost:8000/api/departments
مستندات API: http://localhost:8000/docs


توقف سرویس‌ها:
docker-compose down



نصب دستی

نصب وابستگی‌های Python:
pip install fastapi uvicorn sqlalchemy pydantic


اجرای سرور FastAPI:
uvicorn main:app --host 0.0.0.0 --port 8000 --reload


اجرای فرانت‌اند:

فایل‌های HTML را با یک سرور محلی (مثل Live Server در VS Code) باز کنید.
یا Nginx را به‌صورت دستی تنظیم کنید (فایل nginx.conf را کپی کنید).

📚 API Reference
API با FastAPI پیاده‌سازی شده و مستندات تعاملی آن در /docs قابل دسترسی است. در زیر، نمونه‌ای از endpointها آورده شده است:



متد
مسیر
توضیحات
پارامترها



GET
/api/departments
لیست دانشکده‌ها
limit, offset


GET
/api/students
لیست دانشجویان
limit, offset, department, major, search


POST
/api/students
ایجاد دانشجو
Body: StudentCreate


GET
/api/students/{stid}
اطلاعات یک دانشجو
Path: stid


PUT
/api/students/{stid}
به‌روزرسانی دانشجو
Path: stid, Body: StudentCreate


DELETE
/api/students/{stid}
حذف دانشجو
Path: stid


POST
/api/login
ورود به سیستم
Body: UserLogin

مشکل
راه‌حل



خطای Cannot read properties of undefined
بررسی کنید که majors در repo.js پر شده باشد. تابع loadMajors را دیباگ کنید.


عدم اتصال به API
بررسی docker-compose.yml برای مپ پورت 8000 و لاگ‌های backend.


جدول departments خالی است
دانشکده‌ها را به‌صورت دستی با SQLite اضافه کنید (بخش نصب).


خطای CORS
مطمئن شوید که allow_origins در main.py شامل https://harmakan.ir باشد.
